About embTools

This is just a simple collection of tools that I have found useful to have. embTools is broken up into "modules". You have a module for unit conversions (the most common ones that I use), machine stitch time (can account for trims and trim speed if known), and a simple quoting module to help get a quick quote out to the customer. Can even save the quote (either to file or physical copy) for your records as well by using the "Print" button.

This program is bundled as an Electron app, so it is cross platform. As of right now, it is setup for Windows and Linux platforms, although it can be adapted for those users that are on the Mac platform as well. I had originally started this project as a Firefox extension, but some of the storage methods used (for the notes module) were not as persistent as I would like, so that is the main reason why I moved it to an Electron based application.
